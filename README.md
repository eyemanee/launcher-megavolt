  # Comment installer ?
  Allez dans releases et suivez ces étapes :
  ## Android :
  Téléchargez le fichier .apk puis ouvrez le fichier. Si un message d'erreur apparaît faites cette manipulation : <img width="310" height="162" alt="image" src="https://github.com/user-attachments/assets/d85297ba-0730-45ac-b54b-dc512502f340" />
  ## iOS :
  Téléchargez le fichier .mobileconfig et ouvrez le. Appuiez sur **Accepter** puis ouvrez l'application Réglages et appuiez sur Profil téléchargé et appuiez sur Installer sur les 2 pages et si vous avez un code entrez le quand il serait demandé.
  ## Windows
  Téléchargez le fichier .msi et ouvrez le. Appuiez sur suivant sur les 2 pages.
